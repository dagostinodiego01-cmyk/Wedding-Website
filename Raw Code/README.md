# Wedding Website

This repository contains a single-page HTML wedding website for Diego and Bethany. The main site file is:

- `diego-bethany-wedding.html`

## Purpose

The repository is designed to host a static wedding website that can be published directly on Netlify or any other static site hosting service.

## Deploying to Netlify

1. Create a Netlify account at https://app.netlify.com/signup.
2. Connect your GitHub repository to Netlify.
3. Set the build command to none or leave it blank, since this project is a plain HTML file.
4. Set the publish directory to the repository root.
5. Deploy the site.

## Project Contents

- `diego-bethany-wedding.html`: The wedding website HTML file.
- `extracted-skills-summary.md`: A summary of extracted skill definitions from the archive content.
- `impeccable-main/`: Extracted Impeccable skill archive.
- `taste-skill-main/`: Extracted design taste skill archive.

## Notes

- No additional build tools are required.
- The site can be updated by editing `diego-bethany-wedding.html` and pushing changes to GitHub.
- If using Netlify, the site will publish automatically after pushing updates to the connected branch.
